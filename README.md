# Python Mini Projects

A collection of beginner-friendly Python projects to practice file handling, automation, and working with external libraries.

## Projects

### 📁 1. Image Organizer
Automatically organizes `.jpg` images into a separate folder and renames them sequentially.

**Features**
- Organizes JPG images
- Creates an `images` folder automatically
- Renames files sequentially

**Technologies**
- Python
- os module

---

### 📄 2. PDF Merger
Merge multiple PDF files into a single PDF using the PyPDF2 library.

**Features**
- Merge multiple PDFs
- User-friendly command-line interface
- Generates a single merged PDF

**Technologies**
- Python
- PyPDF2

---

## Requirements

```bash
pip install PyPDF2
```

### 3. 💧 Water Drinking Reminder

A simple Python application that sends desktop notifications at regular intervals to remind users to drink water.

**Concepts Used:**

* Python loops
* `time` module
* `plyer` library
* Desktop notifications

## 🚀 Getting Started

Clone the repository:

```bash
git clone <your-repository-url>
```

Install required dependencies when needed:

```bash
pip install plyer
```

Run a project:

```bash
python <filename.py>
```
